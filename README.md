# HappyTrigger API15

Dalamud API 15 向けのチャットログ連動画像表示プラグインです。

## 機能

- `/popimage` で設定画面を開閉
- チャットログ内のキーワード一致で画像を表示
- ローカル画像 / Web画像URL 対応
- 画像サイズ、表示位置、表示時間を設定可能
- テスト表示対応
- トリガー一覧への保存 / 編集 / 削除
- 表示された画像を左クリックドラッグで位置変更
  - ドラッグ後にマウスを離すと設定を保存
- 設定ウィンドウとタブを緑系カラーに変更

## Build

```bash
dotnet build
```
