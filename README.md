# HappyTrigger API15

Dalamud API 15 向けのチャットログ連動画像表示プラグインです。

## 機能

- `/popimage` で設定画面を開閉
- チャットログ内のキーワード一致で画像を表示
- ローカル画像 / Web画像URL 対応
- 画像サイズ、表示位置、表示時間を設定可能
- テスト表示対応
- トリガー一覧への保存 / 編集 / 削除
- 表示された画像を左クリックドラッグで位置変更
- 内部ログから参照できる機能を追加
- バトルログ / 内部ログ の組み合わせトリガーの追加
- 前提条件トリガーの追加


- テキスト表示トリガーの表示対象テキストをVOICEVOX Engineで読み上げ
  - VOICEVOX Engineが起動していない場合も、トリガー表示は継続します。
  - 音声再生に失敗した場合は、FFXIV Logの内部ログにエラーを出力します。
